# Installs [diyg8](https://github.com/devinsideyou/diyg8) on [Ubuntu](https://www.ubuntu.com/)

```bash
curl -fLOs https://script.install.devinsideyou.com/diyg8
chmod +x diyg8 && ./diyg8
```
